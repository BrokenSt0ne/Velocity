-----
List of console commands:
<b>help</b> <i>This</i>
<b>quit</b> <i>Closes the game</i>
<b>connect <ip> <port> (<password>)</b> <i>Connects to a multiplayer server</i>
<b>disconnect</b> <i>Disconnects from a server</i>
<b>newserver (<password>)</b> <i>Creates a new server and connects the client to it</i>
<b>stopserver</b> <i>Stops the server if you are running it</i>
